網頁主控台：實際運作截圖

以下畫面取自實際啟動的服務。我們把平台本身的原始碼（Python 後端、React／TypeScript 前端、測試與文件， 共 41 個檔案、189 個片段）餵進系統，再逐頁操作擷取。所有數字與搜尋結果都是真實資料，並非示意圖。

本次擷取在無對外金鑰的沙箱環境執行，因此凡需呼叫 Claude 的功能（知識萃取、Ask 問答） 皆會明確回報需要 API key；嵌入、混合檢索與多知識庫切換皆為實際運算。 只要設定 ANTHROPIC_API_KEY，摘要與引用答案便會一併顯示。 畫面左側可見「Knowledge base」切換器（此處有 domain_knowledge 與 web_frontend 兩個知識庫）。
